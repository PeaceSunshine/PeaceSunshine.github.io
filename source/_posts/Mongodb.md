## windos下安装MongoDB

```javascript
1.根目录下创建data文件夹
```



### 命令行下运行MongoDB服务器



```javascript
1.执行mongod.exe文件 ：c：...\mongod   --dbpath    c：...\db

2.连接MongoDB：c:\mongo.exe
```

### 配置MongoDB服务器

```javascript
1. C:\mongodb\bin\mongod.exe    --config    "C:\mongodb\mongod.cfg"   --install
2. net    start   MongoDB
```
